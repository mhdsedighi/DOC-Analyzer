How to use the DOC Analyzer APP:


1-Install Ollama ( from here https://ollama.com/download )

2-Add a Model to Ollama (you can select from the app later) by CMD command.
for example:

ollama run qwen2.5
or:
ollama run deepseek-r1:1.5

(see all Models here: https://ollama.com/library)

3-Insatll Python ( from here: https://www.python.org/downloads )


3-Install required Python packages with CMD command:
pip install ollama PyPDF2 pytesseract pdf2image pillow python-docx openpyxl python-pptx


4-Run the APP:

*** Make sure your VPN or Proxy software is Not Active!

 Click on the 'Run' file or do the following:

       open a CMD in this Folder
       run this command:
       python .\DOC_Analyzer.py

I am actively developing this app. So if you don't know how to work with Git, just click on 'Update App' once in a while to get the latest python file.
